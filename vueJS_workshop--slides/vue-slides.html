What is Vue.js?

  • JavaScript framework for building front-end UI
  • In the same category as React, Preact, Angular and Ember
  • Ability to organize and simplify development

Why Vue?

  • Easy to learn *
  • It's just JavaScript
  • Reactive
  • Components
  • Size
* YMMV

Devtools gotchas *

  • If the page uses a production/minified build of Vue.js, devtools inspection is disabled by default so the Vue pane won't show up.
  • To make it work for pages opened via file:// protocol, you need to check "Allow access to file URLs" for this extension in Chrome's extension management panel.

using vue-cli

[It's like create-react-app]

Vue CLI

  • Install vue-cli (https://github.com/vuejs/vue-cli), 
    								$ npm install -g vue-cli
  • Create project 
    								$ vue init [template-name] [project-name]
  • [template-name] can be webpack || webpack-simple || browserify || browserify-simple || pwa || simple

Vue CLI

  • Start Server 
    								$ npm run dev
  • Go to localhost:8080

Vue CLI (provided project)

  • Open cli--START in your code editor, and cd into project folder in terminal
  • Install packages: 
    								$ npm install
  • Start server: 
    								$ npm run dev
  • Go to localhost:8080

Style scoping (and SASS with node-sass)

						$ npm install sass-loader node-sass --save-dev
